mongoDB安装教程

  1. 下载安装包，并且解压

  2. 在mongodb目录下建好data 和log两个文件夹

    （1）在data文件家中建立db文件夹（放数据库数据）

    （2）在log下面建立mongo.config

  3. mongod.exe –dbpath “E:\Program Files\MongoDB\Server\4.0\data\db” 配置存放文件目录

  4. 可启动mongo.exe 也可在http://localhost:27017/访问

  5. mongod –dbpath E:\MongoDB\data\db –logpath E:\MongoDB\log\mongo.log –logappend –serviceName MongoDB –auth –install 配置wendows server 服务
