  1. Vue_Router

    1. Vue Router是Vue.js官方的路由管理器

      将组件(Component)映射到路由（Routes），告诉Vue Router在哪里渲染他们

      1. 路由
        1. 在实际开发中，App.vue只是作为为一个存放组件的容器
      2. 用映射路由(@指向src目录)

    2. 编程式导航

      1. this.$route.push('/app')

      2. this.$route.push{path: ‘/app’})

      3. a

      4. 获取路由参数：this.$route.params.id

      5. vue路由传参，params和query

        1. query传参（路由里显示参数）
        1. const { msg } = this.$route.query（从路由里面拿数据）
          1. params（路由里不显示参数，会缓存参数，刷新消失）
        2. const { msg } = this.$route.params 刷新页面，数据获取不到（缓存）

      6. destroyed(生命周期)也会删除缓存数据

        1. this.$route.back // 返回上次页面
        2. 保存当前的状态（忽略钩子函数）(提供两个钩子函数activeted、deactivated)
          1. activated
          2. deactivated
