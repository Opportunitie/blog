Webpack 学习

  • 学习知识

    1. webpack插件

      • vue-loader // 处理编译vue
      • extract-text-webpack-plugin // 提取分离css
      • html-webpack-plugin // 将编译后的文件自动插入到index.html中
      • clean-webpack-plugin // 清空dist目录下文件

    2. package.json 脚本

      • “build”: “webpack –config webpack.config.js” // 编译处理文件
      • “watch”: “webpack –watch”, // 文件变化，重新编译
      • “start”: “webpack-dev-server –open”,// 将webpack与提供实时重新加载的开发服务器一起使用,实现热加载
      • “server”: “node dev-server.js” // 通过node启动
