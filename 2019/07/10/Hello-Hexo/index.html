    • 箭头函数this 

      说明：箭头函数没有自己的 this 绑定。箭头函数中使用的 this，
其实是直接包含它的那个函数或函数表达式中的 this，而且箭头函
数在定义的时候已经定了，之后无法改变this指向
const obj = {
test() {
    const arrow = () => {
    // 这里的 this 是 test() 中的 this，
    // 由 test() 的调用方式决定
    console.log(this === obj);
    };
    arrow();
},
getArrow() {
    return () => {
    // 这里的 this 是 getArrow() 中的 this，
    // 由 getArrow() 的调用方式决定
    console.log(this === obj);
    };
}
};

obj.test(); // true
const arrow = obj.getArrow();
arrow(); // true
