Webpack学习

  • webpack

    1. 安装 webpack，以全局的方式安装：

      npm install webpack -g

    2. 进入目录初始化项目

      cd 项目名称

      npm init

      初始化之后会在当前目录下身材package.json 文件

    3. 安装 webpack 依赖

      npm install webpack –save-dev
